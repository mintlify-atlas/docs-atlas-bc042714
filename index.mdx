Rectify11 is a free, open-source Windows 11 modification that restores visual consistency across the operating system. It patches hundreds of legacy system icons, applies custom themes with a UxTheme patcher, and offers optional extras like an enhanced context menu, accent colorizer, and Mica effects — all through a guided installer with a full uninstaller and automatic restore point.

## Get up and running

<Steps>
  <Step title="Check requirements">
    Confirm you are running Windows 11 build 22000 or later and have administrator access. See [system requirements](/system-requirements).
  </Step>
  <Step title="Download the installer">
    Grab the latest release from [GitHub](https://github.com/Rectify11/Installer/releases) and run `Rectify11Installer.exe` as administrator.
  </Step>
  <Step title="Choose your components">
    Select themes, icons, and extras in the installer wizard. A system restore point is created automatically before any changes are made.
  </Step>
  <Step title="Restart your PC">
    After installation completes, restart Windows for all changes to take effect.
  </Step>
</Steps>

<Note>
  Rectify11 V3 is no longer recommended. The team is working on V4, which uses a completely new approach. This documentation covers the V3 installer.
</Note>
